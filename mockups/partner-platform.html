03 · qué tan rápido integran

cuatro líneas.
siete minutos.

una de las razones por las que stripe ganó pagos: era el SDK más rápido de integrar. dovo copia esa jugada. el dev de strava no quiere construir custodia regulada, KYC, dispute resolution, payout — quiere un botón.

cómo lo integra strava

  1. 01 npm install el SDK de dovo (10 segundos)
  2. 02 copia tu API key del partner console
  3. 03 renderiza el <TratoButton /> donde quieras en tu UI
  4. 04 configura un webhook para recibir eventos de tratos cumplidos / fallidos
  5. 05 dovo se encarga del resto: custodia, yield, KYC, payouts, disputes, compliance
strava-app/feed.tsx typescript
import { TratoButton } from '@dovo/react' export function FeedItem({ user, friend }) { return ( <TratoButton apiKey={env.DOVO_KEY} duo={[user.id, friend.id]} goal="run 3× / week" duration="30d" defaultStake={500} onCumplido={handlePayout} /> ) } // dovo se encarga de custodia, yield, KYC, // payouts y compliance. strava solo renderiza.
04 · cómo gana dovo (y por qué un VC paga 30×)

tres revenue streams.
un solo motor.

el mismo SDK que strava integra para "trato fitness" lo integra platzi para "trato educativo" y nu para "ahorro compartido". una sola pieza de infraestructura → múltiples mercados → ingresos compuestos sin tener que construir cada vertical.

stream 01
platform fee
cada trato creado vía la API: $10–30 mxn flat. apps freemium pagan menos, enterprise paga más. el 80% del revenue temprano.
→ $0.5–1.5 mxn × millones de tratos
stream 02
custody revenue
dovo comparte el float que generan las bóvedas con los partners bancarios (nu, klar). aplica fee de gestión sobre el AUM total bajo custodia.
→ 0.8–1.2% sobre AUM
stream 03
trato score API
año 2+: fintechs, dating apps, aseguradoras pagan por consultar el score de cumplimiento de un usuario como signal de underwriting alternativo. moat de datos.
→ por consulta · contratos anuales
narrativa para el deck
"dovo es la capa de commitment infrastructure de latinoamérica. cada app con dos usuarios que comparten un objetivo es un canal de distribución gratis para nosotros. cada banco regulado en mx es un partner de custodia. cada trato cumplido alimenta un dataset único e irreplicable. el moat compone con el tiempo."
